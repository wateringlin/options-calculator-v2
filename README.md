# 📊 期权垂直价差计算器 - 反马丁风控版

一个专为期权卖方设计的风险管理工具，帮助你避免"赚小亏大"的困境。

## 核心理念

**反马丁策略**：与传统马丁策略（亏损加仓）相反，本工具采用：
- 亏损后减仓（保护本金）
- 盈利后恢复仓位
- 严格的每日风险限额
- 连续亏损强制停手

## 功能

### 基础功能
- 输入卖出价和买入价，自动计算净收入
- 自定义止盈止损比例（默认 50% / 200%）
- 实时计算止盈止损目标价差
- 估算单腿目标价格，适配富途 APP 设置
- 移动端优化的深色主题界面

### 风控功能（新增）
- 🚨 **实时风险状态面板**：显示当前本金、今日盈亏、连续亏损次数
- 📉 **反马丁仓位管理**：
  - 连亏 1 次 → 下次开仓 75%
  - 连亏 2 次 → 下次开仓 50%
  - 连亏 3 次 → 停止交易，休息一天
  - 盈利 1 次 → 恢复 100% 仓位
- 💰 **每日风险限额**：设置每日最大亏损比例，到达上限自动停手
- ✅ **开仓检查清单**：5 项检查必须全部通过才能开仓，防止冲动交易
- 📝 **交易记录系统**：
  - 本地存储所有交易
  - 自动计算真实胜率、总盈亏、最大回撤
  - 支持导出 CSV

## 为什么需要反马丁策略？

期权卖方（Credit Spread）的特点是**高胜率、小盈利、大亏损**：
- 胜率可能高达 80-90%
- 每次盈利几百美元
- 但一次大亏可能亏损几千美元

传统马丁策略（亏损加仓）在这种环境下是**致命的**，因为：
1. 期权没有"均值回归"保证
2. 一次黑天鹅事件可以清零账户
3. 越亏越加仓只会加速爆仓

反马丁策略的核心是：**保护本金优先，机会永远有，本金没了就什么都没了。**

## 使用方法

### 开仓流程

1. **检查风险状态**：查看顶部面板，确认"可以开仓"
2. **完成检查清单**：5 项检查必须全部勾选
3. **输入开仓价格**：
   - 卖出期权价格（Short Leg）
   - 买入期权价格（Long Leg）
4. **查看建议仓位**：系统会根据当前状态计算建议合约数
5. **设置止损单**：在富途 APP 中按计算结果设置
6. **记录交易**：平仓后记录盈亏，系统自动调整下次仓位

### 推荐参数设置

- **本金**：5 万美元
- **基础风险比例**：2%（每笔最大亏损 $1,000）
- **每日最大亏损**：3%（$1,500）
- **止盈/止损**：推荐 50%/100% 或 50%/150%

### 示例

- 卖出价: $0.67
- 买入价: $0.17
- 净收入: $0.50
- 止盈 50% → 价差降至 $0.25 时平仓，盈利 $25/张
- 止损 100% → 价差升至 $1.00 时平仓，亏损 $50/张

### 5 万本金建议

| 状态 | 仓位系数 | 每笔风险 | 建议合约数 |
|------|----------|----------|------------|
| 正常 | 100% | $1,000 | 根据止损计算 |
| 连亏1次 | 75% | $750 | 减少25% |
| 连亏2次 | 50% | $500 | 减少50% |
| 连亏3次 | 0% | 停止交易 | 休息一天 |

## 数据存储

所有数据保存在浏览器本地存储（localStorage），包括：
- 交易记录
- 本金设置
- 今日盈亏
- 连续亏损次数

清除浏览器数据会丢失这些记录，建议定期导出 CSV 备份。
