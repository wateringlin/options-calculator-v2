VXM 富途 · 张数计算器

与 vxm期货-日线-多空均做-+spx-trend-1分线.md 对齐：entry_qty_vxm（普通）与共振 min(max(urgent,1), remaining_contracts_cap_*)。佣金 FUTU_VXM_COMMISSION_PER_CONTRACT=4 不参与张数。

公共输入

多头参数 — SLICE_EQUITY_PCT_LONG · MAX_MARGIN_USAGE_LONG

用于 entry_qty_vxm(true) 与 entry_qty_vxm_urgent_long

空头参数 — SLICE_EQUITY_PCT_SHORT · MAX_MARGIN_USAGE_SHORT

用于 entry_qty_vxm(false) 与 entry_qty_vxm_urgent_short

普通分支 — longQty / shortQty

else 分支：buySignal / closeBuySignal / rising_*

longQty = entry_qty_vxm(true)
min(want, cap)，受总占用上限
—
shortQty = entry_qty_vxm(false)
min(want, cap)，受总占用上限
—

共振分支 — longQtyUrgent / shortQtyUrgent

脚本：min(max(entry_qty_vxm_urgent_*(), 1), remaining_contracts_cap_*)，单笔放宽后再用总占用剩余张数封顶

longQtyUrgent — dualBuyUrgent
min(max(urgent_long,1), remaining_contracts_cap_long)
—
shortQtyUrgent — dualExitUrgent
min(max(urgent_short,1), remaining_contracts_cap_short)
—

若 Pine 里已注释卖共振开空，shortQtyUrgent 仅作公式对照。封顶后单笔不再超过「总占用上限 − 已占用」折算的张数，更贴近富途可开仓量（仍与券商实时规则有偏差）。

公式摘要

展开 Pine 等价式 
entry_qty_vxm(isLong):
  want = floor(eq * SLICE% / 100 / MARGIN_PER_CONTRACT)
  cap  = floor((eq * MAX% - |pos|*MARGIN) / MARGIN_PER_CONTRACT)
  return max(0, min(want, cap))

entry_qty_vxm_urgent_long / _short:
  want = floor(eq * SLICE_LONG% 或 SLICE_SHORT% / 100 / MARGIN_PER_CONTRACT)
  urgent = max(1, want)
longQtyUrgent / shortQtyUrgent:
  min(urgent, remaining_contracts_cap_long/short)   // 与 entry_qty 同源 cap
